
LoanSaathiHub - Setup Guide (Supabase + GitHub Pages)

1. Create a Supabase project at https://supabase.com/
2. Get your API keys (project URL and anon/public key)
3. Replace keys in your HTML/JS where mentioned (<!-- Supabase auth integration -->)
4. Create a GitHub repository and upload all these files
5. Use GitHub Pages to host: go to repo > Settings > Pages > Source: main /root

For help: https://supabase.com/docs / https://pages.github.com

Note: Never share your Supabase service role key publicly.
